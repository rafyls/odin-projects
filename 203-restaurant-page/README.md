# Restaurant Page

This project (restaurant-page) is the third project of the [JavaScript course](https://www.theodinproject.com/paths/full-stack-javascript/courses/javascript).

The goal of this project is to create a website for an imaginary restaurant by applying the JavaScript and Webpack concepts learned during the course.
